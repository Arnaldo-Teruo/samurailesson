# 関数を作る際（自作関数）のポイント
## コツ
1. １つの関数に対して機能を最小限にする
1. 機能を最小限にするために細かく関数を作ったりする
1. returnする際は最後でなくとも良い
1. 引数（パラメーター）と返す値（return）についてコメントを添える
## 関数名の注意点
1. 機能が分かりそうな名前にする
1. 長い名前にしない
1. 省略しすぎない
1. 命名規則を統一する
## 引数の注意点
1. 引数は少なめに
1. 引数名（変数名）は分かりやすい名前にする
